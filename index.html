/ terminal multiplexer · v3.x
⌃b prefix · key chord · :cmd command-mode default prefix C-b
Sessionsprefix + key
tmuxstart new session
tmux new -s namenew named session
tmux lslist sessions
tmux a / a -t nattach last / by name
tmux kill-ses -t nkill session
tmux kill-serverkill all sessions
⌃b ddetach client
⌃b Dchoose client to detach
⌃b schoose session (tree)
⌃b $rename session
⌃b ( / )previous / next session
⌃b Lswitch to last session
Windowsprefix + key
⌃b ccreate window
⌃b ,rename current window
⌃b &kill window (confirm)
⌃b wchoose window (tree)
⌃b 0 … 9select window by index
⌃b 'prompt window index
⌃b n / pnext / previous
⌃b llast active window
⌃b ffind window by text
⌃b .move window (idx)
:swap-window -t nswap with window n
:move-window -t s:nmove to session/idx
Panesprefix + key
⌃b %split vertical (left/right)
⌃b "split horizontal (top/btm)
⌃b ocycle through panes
⌃b ;toggle last active pane
⌃b ←↓↑→navigate by direction
⌃b qshow pane numbers
⌃b q 0-9jump to pane number
⌃b { / }swap pane prev / next
⌃b zzoom / unzoom pane
⌃b !break pane → new window
⌃b xkill pane (confirm)
⌃b spacecycle layout presets
⌃b tshow clock

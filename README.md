# tmux cheatsheet

A single-page, single-viewport tmux cheatsheet. Pure static HTML — no build, no dependencies.

## View it

Open `index.html` in a browser, or visit the GitHub Pages deployment.

## What's covered

Eight cards, ~100 bindings and commands:

- **Sessions** — start, list, attach, detach, switch, rename
- **Windows** — create, navigate, rename, move, find
- **Panes** — split, navigate, swap, zoom, kill, layouts
- **Resize & Layout** — directional resize, preset layouts, join
- **Copy Mode (vi)** — motion, selection, search, yank
- **Buffers** — paste, save, load, capture
- **Command & Help** — `:` prompt, config reload, bindings, options
- **Misc & Power** — `send-keys`, `pipe-pane`, env, formatting

## Conventions

- <kbd>⌃b</kbd> — the prefix key (default `C-b`)
- <kbd>key</kbd> — chord after the prefix
- `:cmd` — entered at the tmux command prompt (`⌃b :`)
